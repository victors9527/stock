国内外股票系统，包含IPO新股, VIP抢筹，大宗交易，借贷，指数数据。股票交易持仓，平仓，实时K线数据。

我们的使命是用Java开发世界上最好的、高性能的、安全的、开源的（重点）数字货币交易系统。（请忽略我们的牛逼，因为牛逼吹得多了，可能慢慢就真的牛逼了）
